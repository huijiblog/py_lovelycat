# py_lovelycat
这是基于Python对接可爱猫的微信机器人  
  
  
## 【环境】  
可爱猫4.4 - [下载地址][1]  
win10(其它版本系统请自测)  
Python3.9  
  
## 食用方法：
它是基于Flask写的API对接程序  
先运行可爱猫、登录微信 + 配置好 可爱猫4.4的内置插件"http多语言对" ，然后运行`catMain.py`即可
